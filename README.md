# Spotify-Playlist-Scraper
Scrapes my public playlist off Spotify after granting access token via Spotify Developer Console.

Each song entry is populated with following records:
(1)  Danceability
(2)  Energy
(3)  Key
(4)  Loudness
(5)  Mode
(6)  Speechiness
(7)  Acousticness
(8)  Instrumentalness
(9)  Liveness
(10) Valence
(11) Tempo
(12) Type
(13) ID
(14) Uri
(15) Track_HRef
(16) Analysis_URL
(17) Duration
(18) Time Signature
(19) Added_At
(20) Song Title
(21) Artists
